# kit

A kit for AI agents.

## Installation

**Option 1: Add marketplace then install**
```bash
claude /marketplace add https://github.com/andybons/kit.git
claude /install kit
```

**Option 2: Install directly**
```bash
claude /install https://github.com/andybons/kit.git
```
